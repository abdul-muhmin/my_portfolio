The User Portfolio Management project is a web application built using Django, designed to enable users to create, manage, and showcase their personal portfolios. This application provides a secure and user-friendly platform where users can register, log in, and maintain their individual portfolio entries.

  • Users can register and log in securely using Django's built-in authentication system. Each user has a unique profile and can only manage their own portfolio entries, ensuring data privacy and security.

  • Authenticated users can create new portfolio entries, which may include details such as the project title, description, and creation date. Users can update or delete their portfolio entries as needed, allowing for easy management and upkeep of their portfolios.

  • Each user has a personalized dashboard where they can view a list of their portfolios. Users can view detailed information about each portfolio entry and make updates as necessary. Users can view detailed information about each portfolio entry and make updates as necessary.

  • The application is designed to be responsive, ensuring a seamless experience across various devices, including desktops, tablets, and smartphones.Admin users have special privileges to view and manage all portfolios, providing oversight and the ability to maintain the platform effectively.
